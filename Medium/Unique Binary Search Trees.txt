# Intuition
The problem asks for the number of structurally unique Binary Search Trees (BSTs) that can be formed using values from 1 to n.

This is a classic Catalan number problem. Instead of constructing all possible BSTs, we can directly compute the number using the formula:

Cn = (2n)! / ((n+1)! * n!)

This formula gives the total number of unique BSTs.

---

# Approach
We use the mathematical formula for the nth Catalan number.

Steps:
1. Define a helper function f(n) to compute factorial recursively
2. Compute (2n)! using f(2*n)
3. Compute (n+1)! using f(n+1)
4. Compute n! using f(n)
5. Apply the formula: (2n)! / ((n+1)! * n!)
6. Return the result as an integer

---

# Complexity

- Time complexity:
O(n) — factorial computation takes linear time

- Space complexity:
O(n) — due to recursion stack

---

# Code
```python
class Solution:
    def numTrees(self, n: int) -> int:
        def f(n):
            if n == 1:
                return 1
            return n * f(n - 1)

        a = f(2 * n)
        b = f(n + 1)
        c = f(n)

        return int(a / (b * c))
